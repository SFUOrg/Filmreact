# Filmreact Legacy Monorepo

## Описание проекта
- Учебный pet‑проект, моделирующий миграцию со старого стека: фронтенд на `create-react-app`, бэкенд на `Express + SQLite`.
- Клиентское приложение (`film-legacy`) запрашивает базовую информацию о фильмах из OMDb API и отзывы пользователей из локального API.
- Сервер (`film-api-legacy`) разворачивает SQLite-базу из `data/init.sql`, предоставляет REST-эндпоинты `/api/users` и `/api/posts`.
- Дизайн-макет: https://www.figma.com/design/JLQgQp7JrHm99F3DlKZSFL/Untitled?node-id=87-379&t=v3iClbPmr9DlDg2n-1

## Структура репозитория
- `film-api-legacy/` — бэкенд на Express, инициализация SQLite и REST-эндпоинты.
- `film-legacy/` — React-клиент с отображением фильмов и отзывов, стили в `src/App.css`.
- `film-api-legacy/data/init.sql` — скрипт создания таблиц и демо-данных (пересоздаётся при каждом старте).

## Инструкция по запуску

### Требования окружения
- Node.js версии 18+ (проверено на LTS).
- npm версии 9+ (идёт в комплекте с Node.js 18).
- Свободные порты `3000` (фронтенд) и `5000` (API).

### Запуск бэкенда (`film-api-legacy`)
1. `cd film-api-legacy`
2. `npm install`
3. `npm run dev` — режим разработки с `nodemon` (или `npm start` для простого запуска)
4. Сервер слушает `http://localhost:5000`, база создаётся в `film-api-legacy/db.sqlite`.

### Запуск фронтенда (`film-legacy`)
1. В отдельном терминале `cd film-legacy`
2. `npm install`
3. `npm start`
4. Приложение станет доступно на `http://localhost:3000`

> ⚠️ Перед запуском фронтенда убедитесь, что API уже поднят: компонент `App` запрашивает данные с `http://localhost:5000/api`.
